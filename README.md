# 🚗 Driver Drowsiness Detection System 💡

## 🔍 Overview
The Driver Drowsiness Detection System is a real-time, non-intrusive solution designed to monitor driver fatigue using computer vision techniques. By analyzing eye behavior through a webcam feed, the system detects early signs of drowsiness and triggers alerts to prevent accidents.

This project aims to enhance road safety, especially during long-distance driving.

---

## 🚀 Key Features
- 👁️ Real-time monitoring of driver’s eyes using webcam  
- 😴 Fatigue & drowsiness detection using EAR (Eye Aspect Ratio)  
- 🔊 Instant alert system to prevent accidents  
- ⚡ Fast and lightweight system  
- 🌐 Web-based UI using Streamlit  
- 📱 Progressive Web App (PWA) support  
- 📴 Offline functionality with caching  

---
